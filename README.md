# C++ Fundamentals Practice

This repository contains beginner-level C++ programs developed as part of my self-learning journey in C++ programming.

## Topics Covered
- Arrays and basic data handling
- Functions and program modularity
- Recursion
- Object-Oriented Programming (OOP) concepts
- Basic use of enums, unions, and file handling

## Purpose
The purpose of this repository is to build a strong foundation in C++ by practicing core programming concepts and improving logical thinking and problem-solving skills.

I am currently learning C++ independently and aim to apply these fundamentals to backend development and real-world projects under mentorship.
